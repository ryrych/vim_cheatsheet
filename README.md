How can I swap split window position?

+---------+---------+           +---------+---------+
|         |         |           |         |         |
|         |         |           |         |         |
|         |         |           |         |         |
|         |         |           |         |         |
|   B     |    A    | +------>  |   A     |    B    |
|         | (here)  |           |         |         |
|         |         |           |         |         |
|         |         |           |         |         |
+---------+---------+           +---------+---------+

```
Ctrl-w-J
```

More info:
- You can use any direction key (*capital*) to change position: `h`, `j`, `k`, `l`
- Use `Ctrl-w-r` to *rotate* windows.
